# 🎨 HTML-Quick-Styler

### Professional HTML Page Generator with Auto-Styling built using Python and Gradio

This tool generates **modern, responsive HTML pages automatically** based on a simple description.

---

## 🧰 Features

* Generate complete HTML pages instantly
* Modern responsive design
* Multiple color styles
* Auto-detected page sections
* Live preview
* Built using **Gradio UI**
* Ready-to-use HTML output

---

## 🛠 Technologies Used

* Python
* Gradio
* HTML
* CSS

---

## 🎯 Usage

1. Enter a **Page Title**
2. Write a **Description of sections**
3. Choose a **Color Style**
4. Click **Generate HTML**

The tool will generate a **complete HTML page automatically**.

---

## 🎨 Available Color Styles

* Modern
* Vibrant
* Ocean
* Forest
* Sunset
* Luxury
* Creative
* Minimal

---

## 👀 Example Sections

You can generate pages containing:

* Testimonials

* Footer
